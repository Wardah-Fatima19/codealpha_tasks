📜 Project Overview

This is a simple text-based Hangman Game developed as part of the CodeAlpha Python Programming Internship. The game allows the user to guess a hidden word one letter at a time within limited attempts. It demonstrates the use of loops, conditions, lists, dictionaries, and string handling in Python.

🧠 Key Python Concepts Used

- if-elif conditions
- Loops (while loop)
- Lists and dictionaries
- Functions
- String handling
- User input/output
- Random module

⚙️ How It Works

1. The program selects a random word from a predefined list.
2. A hint related to the word is displayed.
3. The player guesses one letter at a time.
4. Correct guesses reveal the letters in the word.
5. Wrong guesses reduce the remaining attempts.
6. The game ends when:
   - The user guesses the word correctly ✅
   - OR all attempts are finished ❌

▶️ To run the program:

python Hangman_Game.py
