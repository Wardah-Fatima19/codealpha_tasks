📜Project Overview

This is a rule-based chatbot developed as part of the CodeAlpha Python Programming Internship. The chatbot interacts with users through simple text-based commands and responds with predefined replies. It demonstrates the use of conditional statements, loops, functions, and string handling in Python.

🧠Key Python Concepts Used

- if-elif conditions
- Loops (while True)
- Functions
- String handling
- User input/output
- Random module
- Date and time module

⚙️ How It Works

1. The chatbot greets the user.
2. It accepts text input from the user.
3. The chatbot checks the input using predefined conditions.
4. It responds to commands like:
   - hello
   - how are you
   - joke
   - time
   - date
   - bye
5. The chatbot stops when the user types "bye".
▶️To run the chatbot:

python Basic_Chatbot.py
