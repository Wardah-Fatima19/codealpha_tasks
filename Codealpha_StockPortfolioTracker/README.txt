📜 Project Overview

This is a simple Stock Portfolio Tracker developed as part of the CodeAlpha Python Programming Internship. The program allows users to enter stock names and quantities, then calculates the total investment value based on predefined stock prices. It also provides an option to save portfolio details into a text file.

🧠 Key Python Concepts Used

- Dictionaries
- Loops
- Functions
- User input/output
- Basic arithmetic operations
- File handling
- Conditional statements

⚙️ How It Works

1. The program stores predefined stock prices in a dictionary.
2. The user enters stock names and quantities.
3. The program calculates the investment value for each stock.
4. The total portfolio investment is displayed.
5. The user can optionally save the data into a text file named "portfolio.txt". 💾

▶️ To run the program:

python Stock Portfolio Tracker.py
